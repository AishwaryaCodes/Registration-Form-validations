# Registration Form Validations

This project demonstrates a registration form with client-side validation using HTML, CSS, and JavaScript. The form ensures that user inputs meet specific criteria before submission.

**Demo Link:**  https://aishwaryacodes.github.io/Registration-Form-validations/ 

## Features

- **Username Validation:** Checks for a non-empty username within a specific length range.
- **Email Validation:** Ensures the email address is in the correct format.
- **Password Validation:** Validates password strength based on character types and length.
- **Confirm Password Validation:** Verifies that the confirmation password matches the original password.
- **Form Reset:** Allows users to clear the form inputs.

## Technologies Used

- **HTML:** For creating the structure of the registration form.
- **CSS:** For styling the form and validation messages.
- **JavaScript:** For implementing form validation logic and interactions.
